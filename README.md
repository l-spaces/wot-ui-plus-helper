## 📖 项目概述

 🦄  wot-ui-plus 助手插件专为使用 wot-ui-plus + Vue 3.0 + ts 开发的用户设计。
 <br/>
 🦄  因此wot-ui-plus 助手 是一款专为 wot-ui-plus  组件库打造的 VS Code 智能提示插件，旨在提升开发者在使用 wot-ui-plus  组件时的编码体验和效率。该插件通过分析 wot-ui-plus 组件的文档，为开发者提供精准的代码补全、悬停提示和属性验证等功能。


## ✨ 特性

- ⚙️为 wot-ui-plus 组件库提供智能代码补全
- 🚀 支持组件标签、属性、事件的智能提示
- 📖 悬停显示组件和属性的详细文档说明
- 🔍 支持动态属性(:prop)和事件(@event)绑定补全
- 🛡️ 提供组件属性值验证和错误诊断功能
- 📄 支持 Vue 和 HTML 文件


## ❓ 安装

### 方法一：VS Code 扩展市场安装（推荐）

1. 在 VS Code 扩展市场中搜索 `wot-ui-plus 助手`
2. 点击安装

### 方法二：离线安装

1. 下载 `.vsix` 文件 [Gitee 下载](https://github.com/l-spaces/wot-ui-plus-helper/releases)   [GitHub 下载](https://github.com/l-spaces/wot-ui-plus-helper/releases)
2. 在 VS Code 中按 `Ctrl+Shift+P` 打开命令面板
3. 输入 `Extensions: Install from VSIX...`
4. 选择下载的 `.vsix` 文件进行安装
