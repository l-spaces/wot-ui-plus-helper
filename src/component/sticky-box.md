# StickyBox 吸顶容器

## 组件概述
StickyBox 是吸顶容器组件，用于包裹多个 Sticky 组件，实现多个吸顶元素的堆叠效果。它会自动管理内部吸顶元素的位置和层级，确保吸顶元素在滚动时正确堆叠，适用于需要多个吸顶元素的场景。

### 详细功能描述
- 支持包裹多个 Sticky 组件
- 自动管理内部吸顶元素的位置和层级
- 实现多个吸顶元素的正确堆叠效果
- 自动监听容器大小变化，适应动态内容
- 支持自定义样式和类名
- 跨平台兼容

### 适用业务场景
- 多级导航栏吸顶
- 多个筛选条件吸顶
- 复杂页面的多个吸顶元素
- 任何需要多个吸顶元素堆叠的场景

该组件没有对外暴露的方法。
