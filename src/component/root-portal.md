# Root Portal

## 组件概述
Root Portal组件是一个跨平台的内容传送门组件，用于将组件的子内容渲染到应用的根节点上，解决了在复杂组件嵌套场景下的层级溢出和样式隔离问题。

### 详细功能描述
- 支持多端适配：H5、微信小程序、支付宝小程序、App Plus
- 自动根据平台选择最佳实现方案
- 保持子组件的原有功能和样式
- 解决组件嵌套导致的层级问题
- 支持动态内容更新

### 适用业务场景
- 弹出层、模态框等需要覆盖整个应用的组件
- 下拉菜单、气泡提示等需要突破父容器层级限制的组件
- 复杂表单中的动态弹窗
- 需要在应用全局层级显示的通知和提示
